# Global Mental Health Rates Unsupervised Learning Analysis

An unsupervised machine learning analysis of global mental health disorder rates across 195 countries using K-Means and Hierarchical Agglomerative Clustering.

## Dataset
Global Trends in Mental Health Disorder dataset (2017) sourced from Kaggle/Our World in Data.

## Methods
- K-Means Clustering (k=7)
- Hierarchical Agglomerative Clustering with Complete Linkage (k=7)
- t-SNE visualization
- DBSCAN noise detection
